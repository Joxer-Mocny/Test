This test is checking website:
https://www.bart.sk/mam-zaujem-test (sk and en version).Test is made for Cypress.io.

Instalation

npm install cypress

if it is not working you can download it on this website : https://www.cypress.io/

Start the cypress aplication and click on select manualy, choose folder. It will make you and folder cypress and you have to put your test in "testfolder"/cypress/integration/examples.

After click in the cypress aplication on the test you want to do.
